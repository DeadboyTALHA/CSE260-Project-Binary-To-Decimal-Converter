*CSE260 Project: Binary To Decimal Converter*

**Description:** The system will show the decimal value of a binary number. It can show up to 10. Output will be shown through 10 separate LEDs.

**ICs/Gates allowed to use:** AND, OR, NOT, NAND, XOR, X-NOR.

**Sample Input-Output:**  
Input: 0011  
Output: (3 LEDs light up, indicating 3)
